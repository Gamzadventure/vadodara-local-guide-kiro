# Vadodara Local City Cultural & Food Guide

This project is built for the **Kiro Heroes Challenge – Week 5: The Local Guide**.

## Objective
To build a simple AI-powered tool that understands Vadodara’s local culture, food, and daily habits using a custom context file.

## How It Works
- A `.kiro/product.md` file stores Vadodara-specific knowledge
- The Python script reads this context
- User queries are answered using localized information

## Technologies Used
- Python
- Custom Context Engineering
- Kiro Agent Concept

## How to Run
1. Clone the repository
2. Run `python main.py`
3. Ask questions about Vadodara city

## Example Queries
- Suggest famous street food in Vadodara
- Tell me about Vadodara festivals
- Best time to travel in Vadodara
